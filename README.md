- JavaScript (ES2017 == ES8)
- SCSS / HTML5
- Chrome / Edge
- Axios
- Node JS LTS
- Npm 8.11.0
- Redux
- ESLint
- Webpack
- 오픈소스 Editor
- Electron

Back-end
- JAVA 1.8
- Spring Boot 2.7.1
- Spring Security
- MariaDB 5.6.47.0
- Hibernate
